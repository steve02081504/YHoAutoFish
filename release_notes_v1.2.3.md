# YHo AutoFish v1.2.3 稳定性更新

v1.2.3 重点完善鱼饵不足后的自动购买流程，并提升 1600 × 900、1080p、2K 等常见分辨率下的鱼饵商店识别稳定性。建议所有 v1.2.2 用户升级。

## 主要变化

- 新增鱼饵不足自动购买万能鱼饵流程，可在设置中配置购买数量，设置为 0 时关闭自动购买。
- 自动购买数量按 99 的倍数调整，设置页会显示预计需要准备的鱼鳞币数量。
- 修复钓鱼商店记住上次选中商品后，可能直接购买错误鱼饵的问题。
- 优化无上限万能鱼饵商品定位，必须通过完整商品卡与同卡货币特征共同确认。
- 修复 1600 × 900 与 2K 下正确商品详情已出现但被详情文字识别噪声拒绝的问题。
- 优化购买数量最大值、购买按钮、确认弹窗和获得物品弹窗的流程衔接。
- 修复购买完成退出商店后可能多按一次 ESC，导致离开钓鱼初始界面的问题。
- 优化鱼饵不足短时提示检测，降低正常抛竿时误进入自动购买流程的概率。
- 新增鱼饵商店与详情页调试开关，方便反馈不同分辨率下的识别截图。
- 优化标题栏群入口与赞助入口展示，赞助显示可在设置中关闭或重新开启。

## 升级建议

- 经常遇到鱼饵用完后循环卡住、无法继续钓鱼的用户建议升级。
- 使用 1600 × 900、2K 或高 DPI 显示环境的用户建议升级。
- 已经在 1080p 下稳定运行的用户也可以升级，本版本保留原有 1080p 流程，并补充分辨率适配。
