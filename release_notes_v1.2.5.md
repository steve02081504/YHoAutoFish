# YHo AutoFish v1.2.5 稳定性更新

v1.2.5 重点复核自动钓鱼主流程，移除自动购买鱼饵流程，并优化耐力条消失后的结算切换时机。建议所有 v1.2.4 用户升级。

## 更新内容

- 移除鱼饵不足自动购买流程和对应设置入口；程序不再因鱼饵不足提示进入商店或触发购买恢复流程。
- 移除抛竿后鱼饵不足横幅探测，避免误识别后按 R 进入商店、恢复 ESC 退出到错误界面。
- 鱼饵不足时由游戏提示，用户手动补充或装备后再继续钓鱼。
- 优化耐力条消失后的结果切换时机：已稳定溜鱼且完成结果探测后可更快进入结算判定，同时保留短时断帧保护。
- 保留成功结算已记录后的关闭保护，避免重复 ESC 退出钓鱼初始界面。
- 保留月卡每日复位功能，可在设置中开启，按北京时间每日 05:02 执行一次双 ESC 复位。
- 保留标题栏群入口与赞助入口展示优化，赞助显示可在设置中关闭或重新开启。

## 升级建议

- 直接覆盖安装即可，原有 `config.json`、`records.json`、`records.db`、`logs/`、`screenshots/`、`captures/` 和 `.updates/` 不会被发布包内置数据覆盖。
- 若此前手动调整过高级设置，升级后建议先使用现有配置测试一轮，再按实际分辨率和钓鱼点环境微调。
- 若开启月卡每日复位，请确认 05:02 左右游戏窗口可被程序重新聚焦；该功能默认关闭，需要用户主动开启。
